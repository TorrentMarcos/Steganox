STEGHIDE UI

"StegHide" es un programa de esteganografía que puede ocultar datos en varios tipos de archivos de imagen y audio . Las frecuencias de muestra del color, respectivamente, no se cambian; lo que hace que la incrustación sea resistente a las pruebas estadísticas de primer orden.

Steghide funciona desde la consola en Windows y GNU/Linux , y es un proyecto de código abierto que sólo ocupa unos 3Mb descomprimido. Steghide no requiere instalación , por lo que puede usarse en pendrives. Para Windows existe "Steghide UI" - Una interfaz para hacer las cosas más fáciles.

COAGULA

"Coagula" es un sintetizador de imágenes; es tanto un editor de imágenes como un generador de sonido a partir de una imagen. Tiene todas las herramientas útiles para dibujar y crear imágenes . Incluye muchas herramientas específicas que no se pueden encontrar en otro editor de imágenes.

Dado que cada audio se puede representar de forma visual, podemos construir un archivo de audio a partir de una imagen que se reproducirá como su espectro-grama (diagrama de un espectro luminoso o acústico). Plataforma de uso sugerida: Windows . Licencia: Gratuita.

DEEPSOUND 2.0

"DeepSound" es una herramienta de esteganografía y conversor de audio muy fácil de usar con la que podremos ocultar datos en archivos de audio . La aplicación también permite extraer "archivos secretos" directamente de los archivos de audio o pistas de CD de audio.

Para añadir aún más seguridad, DeepSound nos permite proteger los archivos con contraseña para que nadie pueda acceder a nuestros mensajes salvo que nosotros lo permitamos. La aplicación trabaja con los formatos FLAC, MP3, WMA, WAV y APE. Plataforma de uso recomendada: Windows . Licencia: Gratuita.
